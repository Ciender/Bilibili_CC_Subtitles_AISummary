通过b站字幕进行AI总结功能，提供DeepSeek chat和responser模型，采用openai标准json，可以自定义链接、模型、apikey等参数，亦可以支持本地模型。 
总结后的内容开头带时间戳，可以点击跳转。支持调整窗口大小与不透明度。  

使用前请现在脚本开头填入自己的apikey。
